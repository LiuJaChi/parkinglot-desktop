# 停車場輔助管理系統 - 車輛通行記錄管理

## 📋 功能介紹

這是一個為停車場設計的車輛通行記錄導入、查詢和統計分析系統。提供完整的 GUI 界面，支持 Excel 和 CSV 格式的數據導入，以及 PDF 報表導出功能。

### 核心功能

- ✅ **數據導入**：支持 Excel (.xlsx, .xls) 和 CSV (.csv) 格式
- ✅ **靈活查詢**：支持車號、身份、結果等多條件組合查詢
- ✅ **統計分析**：詳細的入場、無法入場、預警統計
- ✅ **PDF 報表**：生成專業的統計和詳細記錄 PDF 報表
- ✅ **媒體展示**：支持顯示車輛照片和影片
- ✅ **數據管理**：JSON 本地存儲，自動保存

## 📊 使用指南

### 1. 導入數據

#### 菜單操作
- **檔案 → 追加匯入**：將新數據添加到現有記錄
- **檔案 → 覆蓋匯入**：用新數據替換現有記錄

#### 快速按鈕
- 點擊 **📤 追加** 或 **🔄 覆蓋** 按鈕

#### 支持的格式

**Excel 格式要求**（.xlsx/.xls）
```
時間              | 車道  | 車號    | 身份 | 結果      | 電子標籤
2024-01-15 08:30 | 車道1 | 粵B1234 | 業主 | 入場      | TAG001
2024-01-15 08:35 | 車道2 | 粵B5432 | 訪客 | 無法入場  | TAG002
```

**CSV 格式要求**（.csv）
```csv
時間,車道,車號,身份,結果,電子標籤
2024-01-15 08:30,車道1,粵B1234,業主,入場,TAG001
2024-01-15 08:35,車道2,粵B5432,訪客,無法入場,TAG002
```

### 2. 查詢記錄

使用查詢面板進行多條件搜索：
- **車號**：輸入完整或部分車號
- **身份**：下拉選擇（業主/訪客/員工等）
- **結果**：下拉選擇（入場/無法入場/預警等）
- 點擊 **🔍 查詢** 執行搜索
- 點擊 **清空** 重置所有條件

### 3. 查看統計信息

- 點擊菜單 **查看 → 統計** 或快速按鈕 **📊 統計**
- 查看以下統計項目：
  - 總計筆數
  - 入場筆數和百分比
  - 無法入場筆數和百分比
  - 預警筆數和百分比
  - 車號統計排行
  - 身份分類統計

### 4. 導出 PDF 報表

#### 方式一：菜單導出
- **檔案 → 導出統計PDF**：生成統計匯總報表
- **檔案 → 導出詳細PDF**：生成詳細記錄報表

#### 方式二：統計窗口導出
- 打開統計窗口後，點擊相應按鈕

#### 報表內容

**統計 PDF**
- 總計、入場、無法入場、預警統計
- 無法入場車號排行（TOP 10）
- 身份分類統計表

**詳細 PDF**
- 所有記錄的詳細信息表
- 自動分頁展示

### 5. 設置照片和影片目錄

- **檔案 → 設置照片目錄**：選擇存放車輛照片的目錄
- **檔案 → 設置影像目錄**：選擇存放車輛影片的目錄
- 或使用快速按鈕 **📁 照片** 和 **🎬 影像**

#### 目錄結構要求

```
照片目錄/
├── 20240115/
│   ├── 粵B12345_入口_001.jpg
│   └── 粵B54321_入口_002.jpg
└── 20240116/
    └── ...

影像目錄/
├── 20240115/
│   ├── 083000-粵B12345_入口.mp4
│   └── 083500-粵B54321_入口.mp4
└── 20240116/
    └── ...
```

## 🔧 故障排除

### PDF 無法生成

**問題**：「缺少 reportlab 庫」或「生成 PDF 失敗」

**解決方案**：
```bash
# 重新安裝 reportlab
pip install reportlab -U

# 確保安裝了中文字體支持
# Windows：系統已自帶
# macOS：若無法生成中文，運行：
fc-cache -fv

# Linux：
sudo apt-get install fonts-noto-cjk
```

### 無法找到中文字體

系統會自動降級為英文字體。若需要中文支持，請確保系統已安裝以下字體之一：
- Windows：simsun.ttc、simhei.ttf、msyh.ttf
- macOS：PingFang.ttc、Songti.ttc
- Linux：NotoSansCJK、DroidSansFallback

### 照片或影片無法顯示

1. 檢查目錄設置是否正確
2. 確保文件名包含車號
3. 確保照片文件名包含「入口」文本
4. 對於影片，確保已安裝 ffmpeg：
   ```bash
   # Windows
   choco install ffmpeg
   
   # macOS
   brew install ffmpeg
   
   # Linux
   sudo apt-get install ffmpeg
   ```

## 📝 數據格式規範

### 字段說明

| 字段名 | 類型 | 說明 | 示例 |
|--------|------|------|------|
| 時間 | String | 通行時間，格式 YYYY-MM-DD HH:MM:SS | 2024-01-15 08:30:00 |
| 車道 | String | 通行的車道標識 | 車道1 |
| 車號 | String | 車牌號碼，不能為空 | 粵B12345 |
| 身份 | String | 車主身份 | 業主/訪客/員工 |
| 結果 | String | 通行結果 | 入場/無法入場/預警 |
| 電子標籤 | String | RFID 標籤編號 | TAG001 |

### 數據驗證規則

- 車號為空、null、NAN、nan 的記錄將被忽略
- 所有必填字段都必須有值
- 時間格式必須為 YYYY-MM-DD HH:MM:SS

## 🎯 常見場景

### 場景 1：導入每日記錄並生成報表

1. 打開系統
2. 點擊 **📤 追加** 選擇當日 Excel 文件
3. 等待導入完成
4. 點擊 **📊 統計** 查看統計信息
5. 點擊 **📊 導出統計PDF** 生成報表

### 場景 2：查找特定車輛記錄

1. 在查詢面板輸入車號
2. 點擊 **🔍 查詢**
3. 在表格中查看該車輛的所有記錄
4. 點擊記錄行查看相關照片和影片

### 場景 3：分析無法入場情況

1. 打開統計窗口
2. 在結果下拉框選擇 **無法入場**
3. 點擊 **🔍 查詢**
4. 查看無法入場車號統計排行
5. 導出詳細 PDF 供進一步分析
